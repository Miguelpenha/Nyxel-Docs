Se você não tem familiaridade com código e não sabe exatamente onde colar a sua Tag Nyxel, a maneira mais fácil e segura de fazer isso é pedindo ajuda para uma Inteligência Artificial (como o ChatGPT, Claude ou até mesmo editores como o Cursor/Windsurf).

Basta copiar o prompt abaixo, substituir os campos entre colchetes pelos seus dados e enviar para a IA da sua preferência.

## Prompt de Instalação Padrão

Copie o texto abaixo:

```text
Atue como um desenvolvedor web sênior especializado na minha base de código atual. 
Instale a plataforma de video commerce (Nyxel) no meu projeto.

Aqui está o script que você precisa inserir:
{COLE AQUI A SUA TAG NYXEL}
```

## O que vai acontecer?
Ela vai te devolver o código exato do seu arquivo pronto com a tag do Nyxel inserida no local correto (geralmente dentro da tag `<head>`).

Dessa forma, você economiza tempo tentando adivinhar como o seu framework específico lida com scripts externos!
