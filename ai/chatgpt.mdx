Se a sua plataforma de site é muito customizada, antiga, on-premise, ou usa uma stack específica (como Next.js, Nuxt, Laravel, Django, Ruby on Rails) e você não tem certeza de onde colar a tag do Nyxel, o jeito mais fácil e rápido de descobrir é **pedindo para uma Inteligência Artificial**.

Você pode usar ferramentas gratuitas como o **ChatGPT**, **v0**, **Claude**, **Perplexity** ou **DeepSeek** para isso.

## O "Super Prompt"

A ideia é simples: você copia o texto (prompt) abaixo, adapta os itens entre colchetes `[]` e cola no seu bot de conversa com a IA.

**Copie e cole este prompt:**

```text
Você é um desenvolvedor especialista em integração de front-end. 

Eu acabei de contratar uma ferramenta chamada Nyxel (uma plataforma de vídeo commerce) e preciso instalar o script deles no meu site para que os vídeos flutuantes funcionem. 

O meu site é construído em [SUA TECNOLOGIA - ex: Next.js 14 App Router / Vtex / Nuxt 3 / React puro com Vite].

Aqui está a tag de script HTML que eu recebi do painel do Nyxel:
<script src="https://admin.nyxel.tech/tag/[MEU-ID-AQUI].js" defer></script>

Por favor, me dê um passo a passo EXTREMAMENTE detalhado de como e em qual arquivo específico eu devo colar ou importar essa tag nessa minha tecnologia para que a tag carregue em todas as páginas globalmente, de forma otimizada para SEO e performance. Me mostre exatamente como o arquivo deve ficar com o script inserido.
```

### Como preencher [SUA TECNOLOGIA]

A Inteligência Artificial precisa de contexto para te dar a resposta certa. Substitua a tag dependendo de como você fez (ou quem fez) o seu site.

| Seu Caso                          | O que digitar no Prompt                       |
| --------------------------------- | --------------------------------------------- |
| Sei o nome da tecnologia          | "Next.js 14 com App Router", "React com Vite" |
| Contratei uma plataforma fechada  | "Vtex IO", "Nuvemshop", "Tray", "Wix"         |
| Fui em uma agência / desenvolvedor| "Foi feito sob medida em PHP Laravel"         |
| Não faço ideia de como foi feito  | "Eu não sei a tecnologia."                    |

### Resolução de Problemas com IA

Se o passo a passo que a IA te der der erro, não desista! Apenas diga para ela:
*"Eu fiz o que você falou e recebi o seguinte erro: [COLE O ERRO AQUI]. Como conserto?"*
