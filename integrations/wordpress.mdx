Integrar o Nyxel no WordPress é simples e você nem precisa tocar em código. Recomendamos o uso de um plugin de gerenciamento de cabeçalho para facilitar o processo.

## Instalando via Plugin (Recomendado)

<Steps>
<Step title="Copie o seu script Nyxel">
  No painel do Nyxel, copie o script de integração disponibilizado para o seu site.
  Exemplo: `<script src="https://admin.nyxel.tech/tag/SEU-ID-UNICO.js" defer></script>`
</Step>

<Step title="Instale o plugin Insert Headers and Footers">
  1. Acesse o painel do WordPress (`/wp-admin`).
  2. No menu lateral, vá em **Plugins > Adicionar Novo**.
  3. Pesquise por **"WPCode – Insert Headers and Footers"**.
  4. Clique em **Instalar Agora** e depois em **Ativar**.
</Step>

<Step title="Cole o script">
  1. No menu lateral, aparecerá uma nova opção chamada **Code Snippets** ou **Trechos de Código**. Clique em **Header & Footer** (Cabeçalho e Rodapé).
  2. Na caixa **Header** (Cabeçalho), cole o script do Nyxel.
  3. Clique em **Salvar Alterações**.
</Step>

<Step title="Verifique se está funcionando">
  Acesse a página inicial do seu site. O Nyxel deve carregar e exibir o seu vídeo flutuante automaticamente.
</Step>
</Steps>

## Opcional: Via Google Tag Manager (GTM)

Se você já utiliza o Google Tag Manager no seu WordPress, pode adicionar o Nyxel por lá:

1. Vá no seu contêiner do GTM.
2. Crie uma **Nova Tag** do tipo **HTML Personalizado**.
3. Cole o script do Nyxel.
4. Defina o **Acionador** para **All Pages** (Todas as páginas).
5. Salve e **Publique** as alterações.
